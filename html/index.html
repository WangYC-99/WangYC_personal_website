    > Tuning Pytorch Training with HUAWEI OpenEuler A-Tune engine

    Training without A-Tune | training using A-Tune

      A-Tune is an OS tuning engine based on AI. I'm extremely surprised by this idea of using operating system to tuning services for the Users. I made an example of using A-Tune to tuning the batch-size, optimizer and learning-rate of the pytorch training (taking the CIFAR-10 DATASET as an example). The result is quite cool that it outperforms the result without tuning by 6.35% in 30 turns.
